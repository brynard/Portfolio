Bookit - Badminton Court Reservation System

BookIt! is a badminton court reservation system developed for the residents of Subang Perdana Goodyear Court 10 as a community service project, for the Application Development course (SECJ 3104).

Project Information

  • Category: Web Application
  • Team Size: 4 members
  • Client: Subang Perdana Goodyear Court 10
  • Project URL: app-dev-bookit
  • Methodology: Agile Scrum development with 4 Sprints
  • Frontend: Bootstrap
  • Backend: Python Flask Framework
  • Database: Google Cloud Platform's BigQuery
